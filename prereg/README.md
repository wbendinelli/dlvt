# prereg/ — Pre-registration materials for the DLVT empirical program

This folder holds the OSF-style pre-registration for the DLVT "killer test":
the 2×2 field experiment (scope reduction β↓ × recovery support R↑) specified
in paper chapter 6 (`paper/chapters/06_empirical_program.tex`, §6.5).
`PREREGISTRATION.md` is a **DRAFT/TEMPLATE** — fields marked `[TO BE
FINALIZED]` must be completed (SESOI, N per arm, protocols, IRB) before
freezing on OSF; it operationalizes propositions P1–P6 of the paper, chiefly
P3 (scope absorption) and P6 (intervention asymmetry), with H3 testing β·C^η
conservation. Power and recovery simulations: companion `dlvt/recovery.py`.
